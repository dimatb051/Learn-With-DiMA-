# Programs in this repository
Here are the small projects I've built to practice these concepts:

1.SumTwoNumbersInput: calculates the summation of two numbers from the user.
2.SumTwoNumbersFixed: a basic program with hardcoded "fixed" values to test arithmetic, to verify the logic. 
3.EvenChecker: a simple logic gate to check if a number is even or odd.
4.OddChecker: specifically checks for odd numbers.
5.PrimeChecker: a bit more advanced—uses trial division to find prime numbers.

# How to Run?
To compile and run any of these files, use the following commands in your terminal:

Compile:
javac FileName.java

Run:
java FileName
